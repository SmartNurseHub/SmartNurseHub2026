# SmartNurseHub2026 🏥

SmartNurseHub2026 คือระบบบริหารจัดการข้อมูลผู้ป่วยและบันทึกทางการพยาบาล  
พัฒนาด้วย **Node.js + Express** และใช้ **Google Sheets เป็นฐานข้อมูลกลาง**  
เหมาะสำหรับหน่วยงานทางการแพทย์ คลินิก หรือโรงพยาบาลขนาดเล็ก–กลาง

---

## ✨ Features
- 📋 จัดการข้อมูลผู้ป่วย (Patients)
- 🩺 บันทึกทางการพยาบาล (Nursing Records)
- 📅 นัดหมายผู้ป่วย (Appointments)
- 🖨️ พิมพ์สติกเกอร์ข้อมูลผู้ป่วย
- ☁️ เชื่อมต่อ Google Sheets API
- 🌐 Web-based ใช้งานผ่าน Browser

---

## 🛠 Tech Stack
- **Backend**: Node.js, Express
- **Frontend**: HTML, CSS, JavaScript
- **Database**: Google Sheets
- **Auth**: Google Service Account
- **Others**: dotenv
